Feedback für Vibecoding-Projekte

Klick. Kommentiere.
Iteriere.

Teile dein Vibecoding-Projekt per Link. Deine Tester klicken direkt auf die Elemente, die sie kommentieren wollen — und du bekommst ihre Kritik prompt-tauglich als Markdown zurück.

Braucht eine CORS-freie URL (GitHub Pages, Netlify, Vercel). Bei CORS-Fehlern → Bookmarklet nutzen.

Features

Feedback, das direkt in den nächsten Prompt fließt

DOM-verankert

Jeder Kommentar ist an einen CSS-Selector + HTML-Snippet gebunden. Kein „irgendwo auf der Seite".

Kategorisiert

Bug, Feature, Design, Copy, Frage oder Lob — direkt beim Kommentieren. Plus Priorität.

Prompt-fertig

Ein-Klick-Export als Markdown mit Selector, Snippet, Kategorie und Priorität. Rein in den nächsten KI-Prompt.

Zero-Setup

Kein Backend, kein Login, kein Signup. Nur ein Link — für dich und deine Tester.

Presentation-Mode

Führe durch das gesammelte Feedback wie durch eine Slideshow. Perfekt für Team-Reviews.

Präzisionsmodus

Ein Toggle zeigt Tester beim Hover Rolle, Größe und Text des Elements — damit die Referenz stimmt.

Filterbar

Filter nach Kategorie mit Live-Counts. Ein Klick, und du siehst nur die Bugs. Oder nur das Lob.

LLM-freundlicher Export

Markdown enthält Element-Rolle, Position, Styles, Kontext-Chain — der Prompt weiß genau, wo er hinschauen muss.

Workflow

In 3 Schritten von Feedback zu Iteration

1

Link erzeugen

Deine Projekt-URL einfügen, Sharing-Link kopieren. Fertig.

2

Tester klicken

Tester öffnen den Link, klicken auf Elemente, hinterlassen Kommentare. Ohne Signup.

3

Markdown exportieren

Aggregiertes Feedback als Markdown herunterladen. Ab damit in deinen nächsten KI-Prompt.

CORS-Bypass

Bookmarklet — läuft auf jeder Seite

Wenn dein Projekt CORS-Fehler wirft oder im Iframe nicht lädt: Das Bookmarklet injiziert sich direkt in deine Seite — ohne Iframe, ohne Server.

VibeFeedback

In Lesezeichenleiste ziehen

  1. Lesezeichenleiste einblenden (⌘⇧B / Strg+Umschalt+B)
  2. Den gelben Link links in die Leiste ziehen
  3. Dein Projekt im Browser öffnen
  4. Bookmark anklicken — VibeFeedback Layer startet
  5. Elemente anklicken, kommentieren, als Markdown exportieren
